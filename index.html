如何新增或修改零食卡片？
1）在原始碼裡找到一整塊 <a class="product-card" ...>...</a>。
2）整塊複製、貼在後面，就是一個新的零食商品。
3）改裡面的 href="..." 成你要跳轉的外部連結（蝦皮、淘寶、MOMO、自家官網、團購表單等）。
4）把 product-image 的 background-image:url(...) 換成你自己的零食圖片檔案名稱。
5）修改標題與描述裡的「繁體中文」與「English」文字即可。

不需要伺服器、不需要資料庫，只用這一個 index.html 檔案，就可以當作你的零食商品橱窗網站。
How to add or edit snack cards?
1) In the source code, find one full block of <a class="product-card" ...>...</a>.
2) Copy and paste that whole block to create a new snack item.
3) Change the href="..." to the external URL you want (Shopee, Taobao, MOMO, your own store, group-buy form, etc.).
4) Update the product-image background-image:url(...) to your own snack image file name.
5) Edit the Traditional Chinese and English text in the title and description.

This single index.html file can be your snack product window site.
